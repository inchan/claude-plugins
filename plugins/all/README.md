# All Plugin

모든 스킬, 에이전트, 명령어를 포함하는 통합 플러그인입니다.

## 포함된 컴포넌트

### Skills (2개)
- **search-core**: 웹 검색 핵심 기능
- **outsourcing-core**: AI CLI 아웃소싱 핵심 기능

### Commands (4개)
- **tdd-team**: TDD 개발 팀 워크플로우
- **search**: 웹 검색 명령
- **doc-update**: 문서 업데이트 자동화
- **outsource**: AI CLI 아웃소싱

### Agents (9개)

#### TDD 개발 팀 (5개)
- **task-planner**: 작업 분석 및 계획
- **test-writer**: 테스트 코드 작성
- **implementer**: 기능 구현
- **refactorer**: 코드 리팩토링
- **reviewer**: 코드 리뷰

#### 전문 에이전트 (4개)
- **engineer**: 범용 엔지니어링 에이전트
- **search-agent**: 웹 검색 전문 에이전트
- **document-updater**: 문서 업데이트 에이전트
- **outsourcing-agent**: AI CLI 아웃소싱 에이전트

## 사용법

```bash
# 플러그인 설치
/plugin install inchan-cc-plugins/all

# 또는 마켓플레이스에서
/plugin
```
