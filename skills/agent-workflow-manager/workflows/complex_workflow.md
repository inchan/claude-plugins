# Complex Workflow Pattern

**Router → Orchestrator → Workers → Evaluator**

## 적용 조건
- Complexity >= 0.7
- 전체 스택 개발
- 다중 컴포넌트 통합

## 실행 흐름

### 1. Router Project Analysis
- 복잡도 확인
- Orchestrator 호출

### 2. Orchestrator Coordination
- Code Analyzer
- System Architect
- Developers (병렬)
- Test Engineer
- Documentation Writer
- Performance Optimizer (선택)

### 3. Integration & Validation

### 4. Final Evaluation
- 프로젝트 레벨 종합 평가

## 예상 소요 시간
1-3시간
