## 1. Claude (Anthropic)

### 기본 정보

- **모델**: Claude 3.5 Sonnet, Claude 3 Opus
- **컨텍스트 창**: 최대 200K 토큰
- **API 키**: Anthropic API Key 필요

### 강점

1. **긴 문맥 이해**
   - 대규모 코드베이스 분석
   - 긴 문서 요약
   - 다중 파일 리팩토링

2. **복잡한 분석**
   - 아키텍처 설계
   - 코드 리뷰 (보안, 성능, 패턴)
   - 심층 기술 분석

3. **신중한 응답**
   - 안전성 우선
   - 명확한 한계 인정
   - 검증 가능한 주장

### 적합한 작업

```
✓ "이 프로젝트의 아키텍처를 분석하고 개선점을 제안해줘"
✓ "이 코드의 보안 취약점을 찾아줘"
✓ "10개 파일을 읽고 중복 로직을 찾아줘"
✓ "이 API 설계의 장단점을 비교 분석해줘"
```

### 부적합한 작업

```
✗ 빠른 코드 생성 (Codex가 더 빠름)
✗ 단순 번역 (Gemini가 더 저렴)
✗ 수학 문제 풀이 (Qwen이 더 정확)
```

## 2. Gemini (Google)

### 기본 정보

- **모델**: Gemini 1.5 Pro, Gemini 1.5 Flash
- **컨텍스트 창**: 최대 1M 토큰 (Flash: 2M)
- **API 키**: Google AI Studio API Key 필요

### 강점

1. **대규모 데이터 처리**
   - 수백 개 파일 동시 분석
   - 대용량 로그 분석
   - 빅데이터 요약

2. **다국어 지원**
   - 100+ 언어 번역
   - 다국어 문서 분석
   - 다국어 코드 주석

3. **비용 효율**
   - Claude/GPT 대비 저렴
   - 대량 작업에 적합

### 적합한 작업

```
✓ "100개 파일에서 TODO 주석 모두 찾아줘"
✓ "이 영어 문서를 한국어로 번역해줘"
✓ "1000줄 로그 파일 분석하고 에러 패턴 찾아줘"
✓ "여러 언어로 작성된 README 파일 통합해줘"
```

### 부적합한 작업

```
✗ 논리적 추론 (Qwen이 더 정확)
✗ 코드 생성 (Codex가 더 전문적)
✗ 보안 분석 (Claude가 더 신중)
```

## 3. Codex (OpenAI)

### 기본 정보

- **모델**: GPT-4 Codex
- **컨텍스트 창**: 8K-32K 토큰
- **API 키**: OpenAI API Key 필요
- **제약**: **Git 저장소에서만 사용 가능**

### 강점

1. **빠른 코드 생성**
   - 함수, 클래스 자동 생성
   - 테스트 코드 작성
   - 보일러플레이트 코드

2. **코드 디버깅**
   - 버그 탐지 및 수정
   - 성능 병목 분석
   - 코드 최적화

3. **리팩토링**
   - 코드 구조 개선
   - 패턴 적용
   - 레거시 코드 현대화

### 적합한 작업

```
✓ "Python Flask REST API 서버 CRUD 예제 작성해줘"
✓ "이 함수의 단위 테스트 10개 작성해줘"
✓ "이 레거시 코드를 TypeScript로 변환해줘"
✓ "이 코드를 함수형 프로그래밍 스타일로 리팩토링해줘"
```

### 부적합한 작업

```
✗ Git 저장소 외부 사용 (제약 사항)
✗ 아키텍처 설계 (Claude가 더 적합)
✗ 대규모 문서 분석 (Gemini가 더 적합)
```

### 중요 제약

⚠️ **Git 저장소 필수**: Codex는 Git 저장소 내에서만 작동합니다.
- 현재 디렉토리가 Git 저장소가 아니면 에러 발생
- `skip_git_repo_check` 옵션은 ai-cli-ping-pong MCP에서 지원하지만 권장하지 않음

## 4. Qwen (Alibaba)

### 기본 정보

- **모델**: Qwen 2.5, Qwen-Math
- **컨텍스트 창**: 최대 128K 토큰
- **API 키**: DashScope API Key 필요

### 강점

1. **수학 문제 해결**
   - 대수, 기하, 미적분
   - 통계 분석
   - 수식 유도

2. **논리적 추론**
   - 알고리즘 설계
   - 최적화 문제
   - 논리 퍼즐

3. **비용 효율**
   - 중국 리전에서 저렴
   - 수학 전문 모델

### 적합한 작업

```
✓ "이 미분 방정식을 풀어줘"
✓ "최단 경로 알고리즘을 설계해줘"
✓ "이 데이터셋의 통계 분석 결과를 계산해줘"
✓ "동적 프로그래밍으로 배낭 문제 풀이해줘"
```

### 부적합한 작업

```
✗ 영어 긴 문맥 이해 (Claude가 더 적합)
✗ 코드 생성 (Codex가 더 빠름)
✗ 다국어 번역 (Gemini가 더 정확)
```

## 선택 가이드

### 복잡도 기반 (향후 v0.2.0)

```
복잡도 낮음 (< 200 토큰):
  - 단순 작업 → gemini, qwen

복잡도 중간 (200-1000 토큰):
  - 코드 작업 → codex
  - 분석 작업 → claude
  - 번역/요약 → gemini

복잡도 높음 (> 1000 토큰):
  - 심층 분석 → claude
  - 대규모 데이터 → gemini
```

### 도메인 기반

```
코드 생성/디버깅 → codex
아키텍처/보안 분석 → claude
대규모 데이터/번역 → gemini
수학/알고리즘 → qwen
```
