## 🎯 워크플로우 관리 스킬

### 1. agent-workflow-manager
- **평점**: ⭐⭐⭐⭐⭐
- **강점**: 전체 워크플로우 자동 조율, Router → Sequential/Parallel → Evaluator 체인
- **개선점**: 복잡도 판단 기준 명확화 필요

### 2. intelligent-task-router
- **평점**: ⭐⭐⭐⭐⭐
- **강점**: 8개 카테고리 분류, 복잡도 기반 모델 선택
- **개선점**: 카테고리 확장 검토

### 3. parallel-task-executor
- **평점**: ⭐⭐⭐⭐
- **강점**: 2-10x 속도 향상, Sectioning/Voting 모드
- **개선점**: 의존성 관리 개선 필요

### 4. dynamic-task-orchestrator
- **평점**: ⭐⭐⭐⭐
- **강점**: 6개 전문 워커 조율, 복잡한 프로젝트에 최적
- **개선점**: 워커 간 통신 프로토콜 표준화

### 5. sequential-task-processor
- **평점**: ⭐⭐⭐⭐
- **강점**: 명확한 단계별 실행, 검증 게이트
- **개선점**: 롤백 메커니즘 추가

### 6. iterative-quality-enhancer
- **평점**: ⭐⭐⭐⭐⭐
- **강점**: 5개 차원 품질 평가, 반복 개선
- **개선점**: 평가 기준 커스터마이징

### 7. agent-workflow-advisor
- **평점**: ⭐⭐⭐⭐
- **강점**: 최적 패턴 추천
- **개선점**: 학습 기반 추천 시스템
